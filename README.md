# Portafolio de Automatización de Pruebas Web - Tomás Lissarrague

Este repositorio alberga los proyectos de automatización de pruebas Web que desarrollé durante la mentoría QA Pro Level de QAX. 
Mi objetivo es demostrar mis habilidades para crear, ejecutar y mantener tests robustos y confiables, garantizando la mejor calidad. 

### Tecnologías que utilizo 
- **Lenguaje:** Java, JavaScript 
- **Frameworks de Automatización:** Playwright 
- **Gestión de Pruebas:** Mocha 
- **Herramientas Adicionales:** Git
